GenArtist Application (10/2024)

This project focuses on collecting and visualizing AI-generated images, generated with various artists to categorize them. It utilizes Supabase, an open-source alternative to Firebase for managing both database and storage. The project is deployed using Vercel, a platform designed for seamless deployment of web applications, similar to Google Cloud Run or AWS Amplify Hosting. FastAPI serves as the API backend, enabling efficient data fetching and integration with the Supabase database and storage. The combination of these tools ensures scalability, fast deployment, and easy management of data and images. This is a project in progress.
